# おとなの学び直し時事英語アプリケーション

## 概要
BBCニュース記事を使って英語学習を支援するElectronアプリです。

## セットアップ
```sh
npm install
```

## 開発
```sh
# React/Electronアプリの起動
npm run dev
```

## テスト
```sh
npm test
```

## ビルド
```sh
npm run build
```

## ディレクトリ構成
- `src/` ... アプリ本体
- `public/` ... HTMLテンプレート
- `docs/` ... 仕様書

## 主な技術
- Electron
- React + TypeScript
- TailwindCSS
- OpenAI/Perplexity API
